# 🎯 우당탕탕 학급 자리 뽑기 웹앱 (Class Seat Picker)

> 학생 명렬을 등록하고, 다양하고 창의적인 게임 방식(랜덤, 룰렛, 뽑기 등)을 통해 공정하고 재미있게 자리를 배치하는 웹 애플리케이션입니다.

---

## 🚀 주요 기능 (Core Features)

### 1. 학생 명렬 관리 (Student Management)
* **텍스트 직접 입력:** 줄바꿈이나 쉼표(,)로 구분하여 학생 이름을 빠르게 등록합니다.
* **엑셀/CSV 업로드:** 대용량 명렬을 한 번에 불러옵니다.
* **저장 및 불러오기:** 브라우저의 `Local Storage`를 활용해 재방문 시에도 명렬이 유지되도록 합니다.
* **특이사항 설정 (옵션):** 시력 제한, 앞자리 고정, 특정 학생 격리(?) 등 최소한의 자리 조정 옵션을 제공합니다.

### 2. 교실 배치도 설정 (Classroom Layout)
* **분단 설정:** 몇 분단, 몇 행, 몇 열로 구성할지 격자(Grid) 구조를 유연하게 설정할 수 있습니다.
* **칠판 위치:** 교실 앞쪽(칠판)을 기준으로 시각적인 템플릿을 제공합니다.
* **사물함/통로 표시:** 빈 공간이나 통로를 지정하여 실제 교실과 유사하게 커스텀 가능합니다.

### 3. 창의적인 자리 뽑기 모드 (Creative Picker Modes)
* **🎲 즉시 랜덤 배치:** 1초 만에 모든 자리를 무작위로 섞어 배치합니다. (가장 빠름)
* **🎡 돌려라 룰렛!:** 한 명씩 이름을 룰렛으로 돌려 당첨된 학생이 원하는 자리(또는 지정된 자리)로 들어갑니다.
* **🃏 카드 뒤집기 (뽑기):** 교실 자리 모양의 카드를 배치해 두고, 학생들이 나와서 카드를 뒤집으면 이름이 등장하는 방식입니다.
* **⚡ 테트리스/드롭 모드:** 화면 위에서 이름 블록이 떨어져 빈자리에 착착 박히는 시각 효과를 제공합니다.

### 4. 결과 저장 및 공유 (Export)
* **이미지 저장 (PNG):** 완성된 배치도를 캡처하여 이미지 파일로 다운로드합니다.
* **프린트 모드:** A4 용지에 깔끔하게 인쇄할 수 있는 CSS 가독성을 지원합니다.

---

## 🛠 기술 스택 추천 (Tech Stack)

* **Frontend:** React.js 또는 Vue.js (컴포넌트 단위 개발 및 상태 관리에 유리)
* **Styling:** Tailwind CSS (빠른 UI 구현) + CSS Keyframes / Framer Motion (애니메이션 효과)
* **State Management:** Zustand 또는 Pinia (학생 명렬 및 좌석 데이터 관리)
* **Libraries:** `html2canvas` (화면 캡처/이미지 저장용)
